**All Cities**
----
  Returns json data about all cities Uzbekistan with longitude and latitude.

* **URL**

  /cities/

* **Method:**

  `GET`

* **Success Response:**

  *  "success": true,
    "code": 200,
    "message": "OK",
    "data": [
        {
            "id": 1,
            "name": "Tashkent",
            "longitude": "69.2667",
            "latitude": "41.3000"
        }, ]
 
* **Error Response:**

  * ...
